```css
/* Attribution System Styles */
.prompt-attribution {
  border-left: 4px solid var(--prompt-color);
  padding: 16px;
  margin: 16px 0;
  background: rgba(0,0,0,0.02);
  border-radius: 0 8px 8px 0;
}

.attribution-header {
  display: flex;
  align-items: center;
  gap: 12px;
  flex-wrap: wrap;
}

.version-badge {
  background: #F3F4F6;
  padding: 2px 8px;
  border-radius: 4px;
  font-size: 12px;
  font-weight: 500;
}

.quality-score {
  display: flex;
  align-items: center;
  gap: 4px;
}

.attribution-actions {
  display: flex;
  gap: 8px;
  margin-top: 12px;
}

.attribution-actions button {
  padding: 6px 12px;
  background: #F3F4F6;
  border: 1px solid #E5E7EB;
  border-radius: 6px;
  cursor: pointer;
  font-size: 14px;
  transition: all 0.2s;
}

.attribution-actions button:hover {
  background: #E5E7EB;
  transform: translateY(-1px);
}

/* Inline attribution */
.ai-generated-content {
  position: relative;
  cursor: help;
  border-bottom: 1px dotted #9CA3AF;
}

.attribution-icon {
  font-size: 12px;
  opacity: 0.5;
  margin-left: 2px;
}

/* Responsive */
@media (max-width: 640px) {
  .attribution-header {
    flex-direction: column;
    align-items: flex-start;
    gap: 8px;
  }
  
  .attribution-actions {
    flex-wrap: wrap;
  }
}
```

This visual guide provides clear examples of how the prompt attribution system will look and function across different contexts and devices.
